# Complete File Structure

All configuration files are located in `/Users/corcoss/code/ship/research/configs/`

```
configs/
│
├── README.md                          # Complete setup and usage guide
├── .gitignore                         # Git ignore patterns
├── package.json                       # Root workspace configuration
├── pnpm-workspace.yaml               # Workspace package definitions
├── tsconfig.json                     # Base TypeScript configuration
│
├── shared/                           # @ship/shared package
│   ├── package.json                  # Shared package config
│   ├── tsconfig.json                 # TS config (extends root)
│   └── src/
│       ├── index.ts                  # Main entry - re-exports everything
│       ├── constants.ts              # Shared constants (HTTP_STATUS, etc.)
│       └── types/
│           ├── index.ts              # Re-export all types
│           ├── user.ts               # User, CreateUserInput, UpdateUserInput
│           └── api.ts                # ApiResponse, ApiError, PaginatedResponse
│
├── api/                              # @ship/api package
│   ├── package.json                  # Express + dependencies
│   ├── tsconfig.json                 # TS config (extends root, references shared)
│   ├── .env.template                 # Environment template (checked in)
│   └── src/
│       └── index.ts                  # Express server with CORS, helmet, routes
│
├── web/                              # @ship/web package
│   ├── package.json                  # React + Vite + dependencies
│   ├── tsconfig.json                 # TS config (extends root, references shared)
│   ├── vite.config.ts                # Vite config with env loading
│   ├── .env.template                 # Environment template (checked in)
│   ├── index.html                    # HTML entry point
│   └── src/
│       ├── main.tsx                  # React entry point
│       └── App.tsx                   # Example app using shared types
│
└── scripts/
    ├── worktree-init.sh              # Initialize worktree (unique ports/DB)
    └── check-ports.sh                # Check active worktrees and resources
```

## File Purposes

### Root Level

| File | Purpose |
|------|---------|
| `README.md` | Complete documentation with setup, usage, troubleshooting |
| `.gitignore` | Ignore node_modules, dist, .env.local, etc. |
| `package.json` | Root scripts (dev, build, worktree commands) |
| `pnpm-workspace.yaml` | Defines workspace packages (api, web, shared) |
| `tsconfig.json` | Base TypeScript config extended by all packages |

### Shared Package (@ship/shared)

| File | Purpose |
|------|---------|
| `package.json` | Package metadata with exports for types |
| `tsconfig.json` | Extends root, sets composite: true |
| `src/index.ts` | Re-exports types and constants |
| `src/constants.ts` | HTTP_STATUS, ERROR_CODES, API_VERSION |
| `src/types/user.ts` | User, CreateUserInput, UpdateUserInput |
| `src/types/api.ts` | ApiResponse, ApiError, PaginatedResponse |
| `src/types/index.ts` | Re-exports all type files |

### API Package (@ship/api)

| File | Purpose |
|------|---------|
| `package.json` | Express dependencies, dev/build scripts |
| `tsconfig.json` | Extends root, references shared package |
| `.env.template` | Environment variable template (PORT, DATABASE_URL) |
| `src/index.ts` | Express server with routes using shared types |

### Web Package (@ship/web)

| File | Purpose |
|------|---------|
| `package.json` | React + Vite dependencies, dev/build scripts |
| `tsconfig.json` | Extends root, references shared, JSX config |
| `vite.config.ts` | Vite config with port and env loading |
| `.env.template` | Environment variable template (VITE_API_URL) |
| `index.html` | HTML entry point for Vite |
| `src/main.tsx` | React root rendering |
| `src/App.tsx` | Example component fetching from API |

### Scripts

| File | Purpose |
|------|---------|
| `worktree-init.sh` | Auto-generate .env.local with unique ports/DB |
| `check-ports.sh` | Display worktree status, ports, databases |

## Generated Files (Not Included)

These files are generated per worktree and should NOT be committed:

```
api/.env.local                # Generated by worktree-init.sh
web/.env.local                # Generated by worktree-init.sh
shared/dist/                  # Built by `pnpm run build:shared`
api/dist/                     # Built by `pnpm run build:api`
web/dist/                     # Built by `pnpm run build:web`
node_modules/                 # Installed by `pnpm install`
.pnpm-store/                  # pnpm content-addressable store
```

## Copy Instructions

To use these configurations in your project:

```bash
# Copy all files to your ship directory
cp -r /Users/corcoss/code/ship/research/configs/* /path/to/your/ship/

# Make scripts executable
chmod +x /path/to/your/ship/scripts/*.sh

# Initialize the project
cd /path/to/your/ship
./scripts/worktree-init.sh
pnpm install
pnpm run build:shared
pnpm run dev
```
